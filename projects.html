Beach Ambassador Website

Aug 2023 - June 2024

This website was created as a part of my summer internship with the University of Wisconsin Sea Grant/Water Resources Institutes. This website is accessible and allows all users to learn about the program, how to get involved, and how to stay safe around our waters. Developing this website taught me enhanced my skills in HTML, CSS, JS, but also gave me experience in the development process of a responsive website. I had to meet with many different people to incorporate their feedback and ideas into the website, until it was complete. The website was eventually moved to WordPress which allows for easier updating and maintenance.

Music Quizzer

May 2023

This website is inspired by the Lyricle music trivia website. It was one of my favorite Wordle variations so when the website was shut down, I was inspired to make my own version. I had little knowledge of CSS at the time so I was focused on functionality, rather than design. I learned how to use APIs (Spotify) to access Spotify information and allow users to customize the game to their own Spotify playlists. Unfortunately, as Spotify prohibits the publishing of music trivia games, so this website is unavailable to the public.

EmptyFridge

March 2023

EmptyFridge is an innovative solution to tackle food waste. It allows users to find tasty recipes to use up leftover ingredients. This was my first Hackathon ever so I was nervous and a bit lost. I was able to gather the data and parse it into a useable, JSON format. This project was also my first time doing web development. I had never touched HTML, CSS, or JS before. This project was awarded the Best Sustainability Hack award by Avanade.
